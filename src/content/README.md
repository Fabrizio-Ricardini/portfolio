# README.md

## portfolio-next

> Dual-mode portfolio: Terminal + Modern UI

### Quick Start
```bash
$ git clone https://github.com/fabrizio/portfolio-next.git
$ cd portfolio-next
$ npm install
$ npm run dev
```

### Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

### Modes
- **Terminal:** Retro CRT terminal with file browser
- **Modern:** Glassmorphism landing page with Aurora FX
